Widget Installation

📦 Add Widget to Product Pages

Add this code to your Shopify theme's product.liquid or main-product.liquid file where you want reviews to appear:

{{ '' }} <div data-reviewbloom data-product-id="{{ product.id }}" ></div>

📋 Installation Steps:

  1. Shopify Admin → Online Store → Themes → Edit Code
  2. Find Sections/main-product.liquid
  3. Paste the code above where you want reviews
  4. Save — Reviews will appear automatically! ✅
